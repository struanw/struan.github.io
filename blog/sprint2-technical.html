What does it mean to display inline vs inline blocks?

17 Feburary 19

Reminder: The display property specifies the display behavior of an element.

Display: inline-block lets one to set a width and height on the element. Therefore the top and bottom margins/paddings are respected, but with display: inline they are not. So the major difference is that 'inline' just puts the element with its atrributes next to each other. See example 1

See example 2 - How the elements are inline with each other but the size or attribute are taken into consideration
