Describe what the DOM is and an example of how you might interact with it.

The DOM represents the current content of the page. Through Devtools you can access information/data that you otherwise, could not as just a user viewing or using (clicking/inputting) a web page. Not only can you use it to change styling of colour and text, you can move items around the page to look at layout. More detail can be obtained by looking at things like how many clicks, who were the users, what outputs arrived from using page. Also things like speed to load and the ability to optimise the page.

Accessing data from arrays and objects.

An array (list) and is written as a list of values between square brackets. You access by specifying which value you require in that array. The first index of an array is zero, not one. So the first element is retrieved with listOfNumbers[0]. You can access object properties in two ways: objectName.propertyName objectName["propertyName"]
